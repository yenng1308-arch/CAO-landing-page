# CAO Landing Page

This repository contains the source code for the **CAO Fine Jewellery Landing Page**, featuring:
1. **Next.js Web Application** (`CAO_web/nextjs-app/`): A modern, interactive Next.js application built with TailwindCSS and TypeScript, incorporating high-quality visual galleries, product story presentation, custom consultations, FAQ, and responsive UI components.
2. **Static HTML/CSS version** (`CAO_web/static-html/`): A clean, lightweight static version of the landing page.
3. **Design & Planning Documentation**:
   - `CAO-UI-UX.md V.md` - Details about UI/UX requirements.
   - `Landing Page Plan - GLAMOROSA Necklace - CAO Fine Jewellery.md` - Strategic design plan for the GLAMOROSA necklace showcase.

## Technologies Used
- Next.js 14+
- React 18+
- TypeScript
- TailwindCSS
- Vanilla HTML/CSS

## Getting Started
To run the Next.js application locally:
```bash
cd CAO_web/nextjs-app
npm install
npm run dev
```
